(async) resolve_getWhoisInfo(resolve, reject, domain)

Perform a whois query for the given domain. Currently (only) used to check pending domaind expiration. IMPORTANT NOTE: The keys and values that are returned by the query are strings that have no rules from registrar from registrar or OS to OS (the implementation of the local whois executable makes a difference). This parser is likely to fail for some registrars.
